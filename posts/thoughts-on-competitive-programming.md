# My Thoughts on Competitive Programming

Recently (as of the start of my freshman year), I've started to get into competitive programming. Specifically [USACO](https://usaco.org/index.php) and [Codeforces](https://codeforces.com/). Its been a really amazing journey, and I'm still not even close to where I want to be within the community. However, despite the short amount of time I have been invested in competitive programming, I've still made some interesting observations that I thought I would share.

1. Competitive Programming is completely different from any other field in Computer Science
   I've been coding since I was in 4th grade. I started out with block coding on code.org, then to python, and eventually I worked my way up to a multitude of programming languages and fields. From Machine Learning, to AI, to Data Structures, to Web Scraping. They all were about the same in my eyes.

However, Competitive Programming requires a new skill that I didn't forsee. It required problem-solving skills.

I have a good background in math, I would say. Ive done some AOPS courses, and I've skipped a couple grades of math in my school, however the unique blend of math, problem solving, and programming that Competitive Programming requires has been a unique, but welcome challenge.

2. Competitive Programming is extremely useful for getting internships and jobs down the line
   From talking to upperclassmen friends who have interviewed for internships, and reading about how to prepare for CS jobs online, the one thing in common, was that interview questions require a specific skillset that you have to practice alot.

I believe that starting Competitive Programming early, and doing it for fun, rather than doing it for some grueling interview process has a ton of value. From what I can tell, Competitive Programming is an amazing investment in your future.

3. The Competitive Programming Community has been welcoming
   The last thing I've noticed since starting Competitive Programming, is the community is really amazing. From the multitude of Discord Servers, to Reddit, to Online Forums. Everybody is trying to help eachother get better. I really think that is a healthy environment, and I admire that.
