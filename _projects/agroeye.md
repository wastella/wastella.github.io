# AgroEye

AgroEye is a web app that uses LLMs to help farmers make informed decisions about their crops using data from the meteorilogical data.

## Background
I worked on this for a NASA hackathon. I was in a team of 2 people, and I did the javascript functionality, while my partner did the UI.
## Technologies used

- Express
- Node
- Javascript
- Google Gemini API
- Meteomatics API
